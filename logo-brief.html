The Forge Icon System
Every mark. One system.

Forge is built from Detroit's first principle: make things that last. Geometric marks. Michigan-first color. One visual language from the company mark to every agent. This is the brand system.

Design Brief

The principles and palette that govern everything below
Principles
01 Detroit in every line. Every line bears weight. Geometry does work. Nothing earns a place here without purpose.
02 The mark IS the idea. Not a picture of the product. A diagram of what it does. When you have to think, we failed.
03 One system. Same visual logic from the company mark to the last agent. You see the coherence before you are told it is there.
04 Inherited, not chosen. Maize. Blue. Wings Red. Tigers Orange. Pistons Silver. These colors carried meaning before we touched them.
05 Monochrome first. Every mark works in one color. Color is the reward for geometry that already stands.

Typography

Two faces. One for reading. One for building.

Display / UI

Forge Co.

Segoe UI Variable · Weight 200–600 · Used for headlines, UI, slides, and all marketing copy.

Code / Mono

$ forge --build

Cascadia Code · All terminal output, code, labels, hex values, and CLI copy.

Usage Rule
01 Headlines: Segoe UI, weight 200. Light carries authority here.
02 Labels and tags: Cascadia Code, uppercase. Letters breathe — slightly open spacing.
03 Never mix weights aggressively. One bold per visual unit.
04 Dark backgrounds only. Light mode is not the primary surface.
